What is Bitcoin?

Bitcoin is a decentralised, digital currency. Bitcoin was first developed in 2009 and has been gaining popularity ever since. The currency is stored on a users computer in a piece of software known as a wallet which can be used to send and receive coins using the network. What is a web wallet?

A web wallet is an website that allows you to store, send and receive coins from your browser. This works in a manner similar to the desktop wallet and has all the advantages of having one, but without having to download the blockchain. With a web wallet, your coins are also accessible from anywhere in the world - simple!
